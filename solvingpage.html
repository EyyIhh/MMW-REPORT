Solving Linear Inequalities

Addition and Subtraction Property

The Addition and Subtraction Property of Inequality states that if you add or subtract the same number to both sides of an inequality, the inequality remains true.
For any real numbers a, b, and c:

  • If a < b, then a + c < b + c
  • If a > b, then a + c > b + c
  • If a < b, then a - c < b - c
  • If a > b, then a - c > b - c

Step-by-Step Procedure

  1. Identify the term to isolate (usually the variable).
  2. Add or subtract the same number from both sides to move constants.
  3. Simplify both sides.
  4. Write the solution.

Example 1

Solve: x + 4 < 10

  1. Subtract 4 from both sides: x + 4 - 4 < 10 - 4
  2. Simplify: x < 6

Example 2

Solve: y - 7 > 2

  1. Add 7 to both sides: y - 7 + 7 > 2 + 7
  2. Simplify: y > 9

Practice Problems

Note: The symbol ≤ means "less than or equal to."

  • Solve: m + 8 < 15
  • Solve: k - 3 > 5
  • Solve: p + 12 ≤ 20

Tip: Always perform the same operation on both sides to keep the inequality balanced.

Multiplication and Division Property

The Multiplication and Division Property of Inequality states that if you multiply or divide both sides of an inequality by the same positive number, the inequality remains true. If you multiply or divide by a negative number, you must reverse the inequality sign.

  • If a < b and c > 0, then ac < bc and a/c < b/c
  • If a > b and c > 0, then ac > bc and a/c > b/c
  • If a < b and c < 0, then ac > bc and a/c > b/c (sign reverses)

Step-by-Step Procedure

  1. Isolate the variable term.
  2. Multiply or divide both sides by the same number to solve for the variable.
  3. If multiplying/dividing by a negative, reverse the inequality sign.
  4. Simplify and write the solution.

Example 1

Solve: 3x > 12

  1. Divide both sides by 3: x > 4

Example 2 (Reverse Sign)

Solve: -2y < 8

  1. Divide both sides by -2 (reverse the sign): y > -4

Example 3

Solve: x/5 ≤ 3

  1. Multiply both sides by 5: x ≤ 15

Practice Problems

Note: The symbol ≥ means "greater than or equal to."

  • Solve: 4a > 20
  • Solve: -5b < 25
  • Solve: c/2 ≥ 7

Tip: Always check the sign of the number you are multiplying or dividing by!

Reversing the Inequality Sign

When you multiply or divide both sides of an inequality by a negative number, you must reverse the direction of the inequality sign. This is because multiplying or dividing by a negative number changes the order of the numbers.

Step-by-Step Procedure

  1. Isolate the variable term.
  2. Multiply or divide both sides by a negative number.
  3. Reverse the inequality sign.
  4. Simplify and write the solution.

Example 1

Solve: -4x > 20

  1. Divide both sides by -4 (reverse the sign): x < -5

Example 2

Solve: -x + 2 < 7

  1. Subtract 2 from both sides: -x < 5
  2. Divide both sides by -1 (reverse the sign): x > -5

Example 3

Solve: -6y ≤ 18

  1. Divide both sides by -6 (reverse the sign): y ≥ -3

Practice Problems

Note: The symbol ≥ means "greater than or equal to."

  • Solve: -2m > 10
  • Solve: -n + 5 < 1
  • Solve: -3p ≥ 9

Tip: Reversing the sign is only necessary when multiplying or dividing by a negative number.
